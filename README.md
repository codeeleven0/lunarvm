# lunarvm
A simple emulated CPU.<br>

# Instruction Set
```
set r1 val
```
<br>
Sets `r1` to `val`.<br>
```
put r1
```
<br>
Prints `r1`.<br>
```
lpc r1
```
<br>
Sets `r1` to current PC.<br>
```
add r1 val
```
<br>
`r1` = `r1` + `val`<br>
```
sub r1 val
```
<br>
`r1` = `r1` - `val`<br>
```
mul r1 val
```
<br>
`r1` = `r1` * `val`<br>
```
div r1 val
```
<br>
`r1` = `r1` / `val`<br>
```
hlt
```
<br>
Halt.
